# Portal PPDI

Aplikasi pengelolaan konten administrasi, website, ziswaf, donasi, program, dan transaksi terintegrasi yang dibangun menggunakan **CodeIgniter 4** dan **Tailwind CSS**.

## Fitur Utama
- **Website Statis Dinamis**: Tambah/Edit halaman profil, mitra, galeri, piagam, struktur dengan mudah bak WordPress.
- **Kabar & Laporan**: Berita, artikel, atau publikasi lain dengan WYSIWYG Editor (*TinyMCE*).
- **Ziswaf & Donasi**: Layanan Jemput Zakat, Kurban Online, hingga manajemen Transaksi API Payment Gateway.
- **WhatsApp Broadcaster**: Integrasi kirim pesan massal untuk laporan kurban/zakat dengan otomatisasi API via Node.JS.
- **Modern & Responsive UI**: Tampilan fully responsive pada mobile device maupun desktop yang bersih dan mulus (Tailwind CSS v3 + Alpine.js/jQuery).

## Instalasi
1. Clone / Upload ke direktori.
2. Setup database dari *export sql* jika dibutuhkan.
3. Jalankan `composer install` jika diperlukan pembaruan dependensi.
4. Buat file `.env` dan konfigurasikan *database* serta `app.baseURL`.
5. Buka via browser.
